## 웹 개발 환경 이해하기
### 역사 (History)

#### HTML
- 1993년 (25년 전)
- Tim Berners LEE
- APPLE 2 (문서도구), IBM (문서도구)
- 상호호환성 X (APPLE 확장자를 IBM으로 바꾸는 팀)
- HTML을 만듬, HTTP 기초 설계도 함
- HTML 1.0 (1993년)
  - 문서를 서로 공유하기 위해 만들어짐
  - 문서로써의 웹
  - A 요소 (링크)
- HTML 2.0
- HTML 3.2
- HTML 4.01 (1999년)
- HTML 5 (2007년쯤부터 화두, 2014년)
    - 어플리케이션(프로그램)으로써의 웹 (GMail, Google의 대부분의 Prodcut)
    - 모바일 웹 (2007년 아이폰 출시)
    - 스티브 잡스가 HTML5가 미래다
    - HTML은 원래 쉽게 작성되어있는 언어
- HTML 5.1 (2016년)
- HTML 5.2 (2017년, 출시 중)

#### CSS
- 1996년
- HTML을 이쁘게 만들기 위해서
- 웹에서 비주얼(눈에 보이는 것)은 전부 다 CSS로 작성
- CSS 1 (1996년)
- CSS 2 (1998년)
- CSS 2.1 (2014년)
- CSS 2.2 (ing)
    - CSS의 종합 타래
- CSS 3 (ing)
    - CSS의 각 스타일별로 구분해서 관리
    - FONT
    - LAYOUT
